## 📌 About the Project
This is a beginner-friendly real-time weather application that fetches live weather data using an API and displays it in a clean and user-friendly interface.

This project marks my first step into web development and helped me understand how real-world applications work.

---

## 🚀 Features
✔️ Search weather by city name  
✔️ Displays temperature, humidity, and wind speed  
✔️ Real-time API data fetching  
✔️ Simple and responsive UI  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- Weather API
